## 📖 如何使用本文档

### 对于新人
1. 先阅读**会议纪要**的"产品介绍"章节，了解产品是什么
2. 再阅读**新人上手指南**的"环境搭建"章节，完成开发环境配置
3. 遇到问题时，使用**问题快速索引**定位解决方案

### 对于项目负责人
1. 重点阅读**会议纪要**，了解项目决策过程和风险
2. 参考**经验手册**，提取可复用的方法论
3. 使用文档中的模板，快速启动新项目

### 对于产品经理
1. 阅读**会议纪要**的"需求阶段"和"商业分析"章节
2. 参考**经验手册**的需求管理和商业化运营经验
3. 了解竞品分析和用户增长策略

### 对于开发人员
1. 阅读**新人上手指南**，了解技术架构和开发规范
2. 参考**经验手册**的技术选型和架构设计经验
3. 查阅**experience/**目录的踩坑经验
