# 书籍详情页中世纪风格升级设计方案

## 一、设计概述

### 核心目标
将 Demo 目录下的书籍详情页升级为更符合中世纪古书风格的界面，提升沉浸感和视觉一致性。

### 设计决策记录

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 章节目录布局 | 古典目录表风格 | 最符合中世纪书籍的真实感 |
| 卡牌比例 | 统一 3:4 比例 | 符合炉石传说卡牌美学，视觉一致性强 |
| 章节入场动画 | 书页翻动效果 | 与整体书籍主题呼应 |
| 卡牌交互动画 | 卡牌抽出效果 | 模拟真实抽牌体验，增强游戏感 |
| Tab切换动画 | 羊皮纸翻页 | 与书页翻动效果呼应，体验连贯 |
| 章节状态区分 | 装饰区分 | 蜡封印章等元素增强中世纪氛围 |

---

## 二、章节目录设计

### 布局样式

**古典目录表风格** - 每章一行，罗马数字编号，虚线连接

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  Chapter I   ···············  The Beginning  ✓  │
│                                                 │
│  Chapter II  ···············  The Journey    ✓  │
│                                                 │
│  Chapter III ···············  The Mountain   📌 │
│                           (墨水滴效果)           │
│                                                 │
│  Chapter IV  ···············  The Dragon    🔴  │
│                           (蜡封印章遮盖)         │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 样式规范

| 元素 | 样式 |
|------|------|
| 章节编号 | Cinzel 字体，罗马数字，金色 `#D4AF37` |
| 虚线 | 点状虚线，颜色 `rgba(139, 90, 43, 0.5)` |
| 章节标题 | Cinzel 字体，深棕色 `#2a1810` |
| 行高 | 每行约 40px，章节间有足够呼吸空间 |
| 背景 | 羊皮纸纹理 `#f4e4bc` |

### 章节状态装饰

| 状态 | 装饰效果 | CSS实现要点 |
|------|----------|-------------|
| **已读 (read)** | 章节号旁显示金色 ✓ 标记 | `::after` 伪元素添加图标 |
| **当前 (current)** | 标题旁显示墨水滴图标，轻微脉动发光 | 动画 + 发光效果 |
| **锁定 (locked)** | 整行被半透明蜡封印章遮盖，显示 🔒 | 覆盖层 + 蜡封样式 |

## 三、卡牌设计

### 统一比例

**3:4 比例（炉石传说标准）**

| 卡牌类型 | 尺寸 | 边框颜色 | 特殊效果 |
|----------|------|----------|----------|
| 角色卡牌 | 150px × 200px | 根据角色类型 | 金色/红色/银色光晕 |
| 情节卡牌 | 150px × 200px | 深红色 `#8B0000` | 红色光晕 |

### 角色卡牌边框颜色

| 角色类型 | 边框颜色 | 光晕效果 |
|----------|----------|----------|
| 主角 (protagonist) | 金色 `#FFD700` | 金色光晕 |
| 反派 (antagonist) | 深红 `#8B0000` | 红色光晕 |
| 配角 (supporter) | 银色 `#C0C0C0` | 银色光晕 |
| 路人 (npc) | 灰色 `#666666` | 无光晕 |

### 卡牌布局

- 一行展示 3-4 张卡牌（根据屏幕宽度响应式调整）
- 卡牌间距：20px
- 使用 `display: grid` + `grid-template-columns: repeat(auto-fit, minmax(150px, 1fr))`

## 四、动画效果规范

### 章节目录动画

| 动画名称 | 触发时机 | 效果描述 | 时长 |
|----------|----------|----------|------|
| pageFlipIn | 页面加载 | 书页从右侧翻入 | 0.8s |
| chapterHover | 悬停章节 | 整行背景高亮，虚线变实线 | 0.3s |
| inkDropPulse | 当前章节 | 墨水滴图标脉动发光 | 2s infinite |

### Tab切换动画

| 动画名称 | 触发时机 | 效果描述 | 时长 |
|----------|----------|----------|------|
| pageFlipOut | 切换离开 | 当前内容像书页翻走 | 0.4s |
| pageFlipIn | 切换进入 | 新内容从右侧翻入 | 0.6s |

### 卡牌交互动画

| 动画名称 | 触发时机 | 效果描述 | 时长 |
|----------|----------|----------|------|
| cardDraw | 悬停卡牌 | 上移 15px + 逆时针旋转 3° | 0.3s |
| cardModalIn | 点击卡牌 | 卡牌放大居中，背景暗化 | 0.4s |
| cardModalOut | 关闭详情 | 卡牌缩小回原位 | 0.3s |

### CSS动画代码示例

```css
/* 书页翻入动画 */
@keyframes pageFlipIn {
  0% {
    transform: rotateY(-90deg);
    opacity: 0;
  }
  100% {
    transform: rotateY(0deg);
    opacity: 1;
  }
}

/* 卡牌抽出效果 */
@keyframes cardDraw {
  0% {
    transform: translateY(0) rotate(0deg);
  }
  100% {
    transform: translateY(-15px) rotate(-3deg);
  }
}

/* 墨水滴脉动 */
@keyframes inkDropPulse {
  0%, 100% {
    box-shadow: 0 0 5px rgba(139, 90, 43, 0.5);
  }
  50% {
    box-shadow: 0 0 15px rgba(139, 90, 43, 0.8);
  }
}
```

## 五、整体布局

### 桌面端布局（>1024px）

```
┌─────────────────────────────────────────────────────────────┐
│  [导航栏]                                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────┐   ┌─────────────────────────────┐ │
│   │                     │   │  书籍信息头部               │ │
│   │   章节目录          │   │  + Tab切换按钮              │ │
│   │   (古典目录表)      │   ├─────────────────────────────┤ │
│   │                     │   │                             │ │
│   │   Chapter I ...     │   │   内容区域                  │ │
│   │   Chapter II ...    │   │   (角色卡牌/情节卡牌)       │ │
│   │   Chapter III ...   │   │                             │ │
│   │   Chapter IV ...    │   │   ┌───┐ ┌───┐ ┌───┐        │ │
│   │                     │   │   │   │ │   │ │   │        │ │
│   │                     │   │   └───┘ └───┘ └───┘        │ │
│   └─────────────────────┘   └─────────────────────────────┘ │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 响应式断点

| 断点 | 布局调整 |
|------|----------|
| 移动端 (<768px) | 单列布局，章节目录和内容区垂直堆叠 |
| 平板 (768-1024px) | 双列布局，章节目录占 1/3，内容区占 2/3 |
| 桌面 (>1024px) | 完整双页书籍布局 |
