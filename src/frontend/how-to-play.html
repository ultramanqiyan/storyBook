🧩 Puzzle Gameplay

Each chapter contains a puzzle. Solve it to earn rewards!

Puzzle Rules

🎯 Each chapter has exactly 1 puzzle.
🔢 Maximum 3 attempts.
❌ Wrong answers show remaining attempts.
✅ Correct answer earns 1 random plot card.

Puzzle Rewards

🃏 Random Plot Card x1

Note: Only logged-in users can earn card rewards.
