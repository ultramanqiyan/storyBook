## 2. Narrative Style

### 2.1 Perspective
- **Point of View**: First-person
- **POV Character(s)**: Anna Reeves
- **POV Consistency**: Single, consistent throughout

### 2.2 Narrative Structure
- **Dual Structure**: Yes
- **Structure Type**: Present timeline + Past memories (always present, never fading)
- **Interweaving Method**: Current events trigger vivid memories; the past is as present as the now

### 2.3 Language Style
- **Overall Tone**: Exhausted, desperate, philosophical, ultimately accepting
- **Sentence Characteristics**: Long, detailed descriptions of memories; short, urgent present moments
- **Vocabulary Level**: Moderate to advanced (psychological terms, emotional vocabulary, temporal concepts)
- **Unique Stylistic Elements**: Memories described with the same vividness as present events; the blurring of past and present

### 5.1 Core Themes

3. **The Nature of Time**: Is time linear, or is it constructed by memory?
