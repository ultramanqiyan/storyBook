# Book Specification: The Whispering Network

## Basic Information
- Book Title: The Whispering Network
- Genre: Fantasy / Science Fiction
- Target Audience: All ages
- Theme: Collective intelligence, memory and legacy

## Narrative Style
- Perspective: Third-person limited (Aria's POV)
- Language Style: Contemporary, accessible prose with fantasy and sci-fi elements
- Sentence Characteristics: Medium length, clear expression
- Unique Elements: Collective consciousness, memory network

## Emotional Tone
- Overall Atmosphere: Wonder, discovery, responsibility
- Emotional Arc: Discovery → Connection → Responsibility → Legacy
- Emotional Pacing: Gradual build, accelerating toward resolution

## Character Voices

### Aria (Protagonist)
- Speech Characteristics: Curious, connected, responsible
- Word Choice: Questions, connections, growing awareness
- Personality Traits: Curious, connected, responsible
- Internal Monologue Style: Wondering, connecting
- Unique Expression: Uses network metaphors

### The Network
- Speech Characteristics: Collective, wise, patient
- Word Choice: We, us, together, memory
- Language Evolution: From alien presence to collective wisdom
- Unique Expression: Speaks as collective consciousness

### The Antagonist
- Speech Characteristics: Corporate, dismissive, calculating
- Word Choice: Profit, control, resource
- Personality Traits: Corporate, dismissive, calculating
- Unique Expression: Views network as resource to exploit

## Theme Depth
- Core Theme: Collective Intelligence
- Philosophical Exploration: What is the power of shared knowledge?
- Real-world Metaphor: Collective intelligence and shared knowledge

## World Building
- Setting: Near-future Earth with hidden network
- Technology: Neural networks, collective consciousness
- Social Structure: Hidden practitioners, corporations, guardians
- The Network: Ancient, vast, interconnected

## Sensory Detail Requirements
Each chapter must include at least 3 types of sensory details:
- Visual: Specific colors, light, shapes
- Auditory: Specific sounds, tones, rhythms
- Tactile: Temperature, texture, pressure
- Olfactory: Specific scents, memory smells
- Gustatory: Specific tastes, textures

## Emotional Depth Design
Each emotional beat must include three layers:
1. Surface Reaction - External behavior
2. Internal Activity - Thinking process
3. Deep Motivation - Why this reaction

## Prohibited Elements
### AI Writing Patterns (Must Avoid)
- "And somewhere..." ending
- "Neither of them knew..."
- "will never be the same"
- "just the beginning"
- Rule of Three structure
- Abstract emotional descriptions
- Template dialogue
- Repetitive sentence structures

### Other Prohibitions
- Overly modern tech terminology
- Simple good vs. evil opposition
- Overly smooth solutions

## Open Ending Design (Chapter 10)
- Unresolved Questions: How far does the network extend? What other memories exist?
- New Suspense: Aria's continuing journey
- Character New Beginning: Aria as guardian of the network
- Reader Imagination: What would you do with collective knowledge?

## Quality Check Standards
Each chapter must pass these checks:
1. AI trace detection (all passed)
2. Sensory detail detection (at least 3 items)
3. Emotional depth detection (three layers complete)
4. Character voice detection (uniqueness)
5. Narrative style detection (consistency)
