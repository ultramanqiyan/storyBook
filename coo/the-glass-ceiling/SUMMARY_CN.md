# 《玻璃天花板》中文概要

## 书籍信息

| 项目 | 内容 |
|------|------|
| **书名** | 玻璃天花板 |
| **类型** | 职场剧情 / 心理惊悚 |
| **难度** | 高 |
| **目标受众** | 18-25岁，欧美读者 |
| **语言** | English |
| **章节数** | 10章 |

## 核心设定

Alex Chen以为加入Meridian Tech——硅谷最负盛名的公司——是梦想职业的开始。但在这光鲜亮丽的表面之下，是精心设计的心理操控网络。当Alex在公司中晋升时，他们开始质疑一切：自己的能力、自己的价值，最终是自己的理智。

当他们发现受害者不止自己时，他们必须决定：是要继续玩这场游戏，还是将一切都公之于众。

## 主角：Alex Chen

| 项目 | 描述 |
|------|------|
| **名字** | Alex Chen |
| **年龄** | 24岁 |

| **职位** | 初级产品经理 |
| **核心冲突** | 成功欲望 vs. 操控意识觉醒 |
| **旅程** | 天真新人 → 质疑员工 → 觉醒幸存者 |

|------|------|------|

## 职场PUA策略

| 策略 | 描述 |
|------|------|
| 爱情炸弹 | 招聘和入职期间过度赞美和关注 |
| 贬低 | 旨在破坏信心的微妙批评 |
| 煤气灯效应 | 否认现实，让员工质疑自己的感知 |
| 孤立 | 切断员工与支持网络的联系 |
| 间歇性强化 | 不可预测的奖励创造依赖 |
| 不断提高目标 | 不断变化的期望确保失败 |

## 主题

- **心理操控**: 职场文化如何变得有毒
- **身份与价值**: 将自我与职业成就分离
- **美国梦**: 精英管理神话的阴暗面
- **权力动态**: 系统如何延续不平等
- **抵抗与疗愈**: 操控之后如何找回力量
