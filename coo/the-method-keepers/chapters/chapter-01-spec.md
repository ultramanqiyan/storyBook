### 6.1 Dialogue Segments

| 3 | Elena, System | Seeking explanation, denial | Shock, disbelief | System has authority | Over-explanation |

### 6.2 Dialogue Segment Specifications

#### Segment 1: Morning Conversation (Elena, Sofia)

**Participants**: Elena Vasquez, Sofia Vasquez

**Surface Content**: Morning routine, getting ready for classification

**Subtext**: Both are nervous about the classification, but neither wants to admit it

**Emotional Undercurrent**:
- Elena: Maternal anxiety masked by professional confidence
- Sofia: Teenage indifference masking fear

**Power Dynamics**: Equal, but Elena tries to project confidence

**Dialogue Beats**:
1. Elena asks if Sofia is ready (attempting reassurance)
2. Sofia responds with teenage shrug (deflection)
3. Elena offers to explain the process again (maternal concern)
4. Sofia declines, says she understands (independence)
5. Brief silence, both pretending normalcy

**Action Beats Required**: Coffee cup, backpack, door opening

**What Must Be Revealed**: Sofia's nervousness beneath indifference

**What Must Be Concealed**: Elena's own anxiety

#### Segment 2: System Explanation (Elena, Robert)

**Participants**: Elena Vasquez, Dr. Robert Chen

**Surface Content**: Elena explaining the methodology stratification system

**Subtext**: Elena's pride in her creation, Robert's genuine belief in the system

**Emotional Undercurrent**:
- Elena: Professional pride, genuine belief
- Robert: Respect for Elena's work, shared values

**Power Dynamics**: Elena as expert, Robert as respectful colleague

**Dialogue Beats**:
1. Robert asks about a specific algorithm (genuine curiosity)
2. Elena explains with enthusiasm (professional pride)
3. Robert nods, asks follow-up (shared belief)
4. Elena elaborates on the elegance of the design (deep investment)
5. Both express satisfaction with the system (shared faith)

**Action Beats Required**: Screen gestures, coffee cups, office setting

**What Must Be Revealed**: The logic of the system, Elena's investment

**What Must Be Concealed**: Any doubt about the system

#### Segment 3: Classification Results (Elena, System)

**Participants**: Elena Vasquez, Meridian System Interface

**Surface Content**: Elena receiving Sofia's classification

**Subtext**: Elena's denial, system's clinical detachment

**Emotional Undercurrent**:
- Elena: Shock, disbelief, growing fear
- System: Clinical, helpful, unaware of emotional impact

**Power Dynamics**: System has authority over classification

**Dialogue Beats**:
1. System announces classification is complete (clinical)
2. Elena asks to see results (controlled anxiety)
3. System displays "Methodology-Deficient" (clinical)
4. Elena asks for verification (denial)
5. System confirms, offers explanation (helpful but inflexible)

**Action Beats Required**: Screen interaction, trembling fingers

**What Must Be Revealed**: Sofia's classification, system's inflexibility

**What Must Be Concealed**: Whether this is an error or feature
