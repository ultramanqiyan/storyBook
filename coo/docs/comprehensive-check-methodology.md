# 全面检查方法论

## 检查范围
- 25本书籍
- 250个章节
- 不基于之前的任何结论
- 独立、客观、全面的评估

## 检查维度

### 1. 内容质量
- [ ] 情节逻辑性
- [ ] 人物一致性
- [ ] 世界观完整性
- [ ] 对话自然度
- [ ] 情感真实性

### 2. 技术实现
- [ ] 三层情感模型应用
- [ ] 感官细节数量和质量
- [ ] 对话标签多样性
- [ ] AI写作模式检测
- [ ] 紧张感词汇使用

### 3. SEO优化
- [ ] 章节标题优化
- [ ] 关键词密度
- [ ] 元描述质量
- [ ] 内容结构

### 4. 读者体验
- [ ] 开篇吸引力
- [ ] 节奏控制
- [ ] 悬念设置
- [ ] 结尾效果

## 评分标准
- A: 优秀 (90-100分)
- B: 良好 (80-89分)
- C: 合格 (70-79分)
- D: 需改进 (60-69分)
- F: 不合格 (<60分)

## 输出格式
每本书将生成独立报告，包含：
1. 总体评分
2. 各维度评分
3. 具体问题列表
4. 改进建议
