# 完整质检标准设计方案

## 一、内容质量维度 (权重20%)

### 1.1 内容完整性
- [ ] 所有章节完整，无缺失
- [ ] 每章字数达标（至少1500字）
- [ ] 开头结尾完整，无截断
- [ ] 情节连贯，无逻辑断裂

### 1.2 情节连贯性
- [ ] 主线清晰，情节推进合理
- [ ] 伏笔有回收，无遗漏
- [ ] 时间线一致，无矛盾
- [ ] 因果关系合理

### 1.3 角色一致性
- [ ] 主角性格发展合理
- [ ] 配角设定一致
- [ ] 对话符合角色身份
- [ ] 行为动机可信

## 二、AI痕迹检测 (权重20%)

### 2.1 高频词汇检测
**禁止词汇（出现即扣分）：**
- tapestry, symphony, dance, mosaic, kaleidoscope
- delve, explore, navigate, unlock, harness
- Furthermore, Moreover, Additionally, Consequently

**限制词汇（超过阈值扣分）：**
- 同一形容词/副词在500字内重复3次以上
- 同一过渡词在1000字内重复5次以上

### 2.2 模板句式检测
**禁止句式：**
- "will never be the same"
- "just the beginning"
- "neither of them knew"
- "little did they know"
- "everything had changed"
- "The air carried the faint scent of..."
- "hit me/him/her like a physical blow"
- "For a long moment"
- "The question hung in the air"

### 2.3 结构模式检测
- [ ] 段落开头多样化（避免连续3段以相同结构开头）
- [ ] 章节结尾多样化（避免多章以相同句式结尾）
- [ ] 情感描写多样化（避免重复比喻）

## 三、语言质量维度 (权重15%)

### 3.1 语法正确性
- [ ] 无语法错误
- [ ] 标点使用正确
- [ ] 时态一致
- [ ] 主谓一致

### 3.2 词汇丰富度
- [ ] 避免重复用词
- [ ] 专业术语使用准确
- [ ] 形容词/副词多样化
- [ ] 动词选择精准

### 3.3 句式多样性
- [ ] 长短句搭配合理
- [ ] 简单句与复合句交替
- [ ] 避免连续使用相同句式

## 四、情感深度维度 (权重15%)

### 4.1 三层情感结构
- [ ] 表面反应（可观察行为）
- [ ] 内心活动（想法感受）
- [ ] 深层动机（价值观/恐惧/渴望）

### 4.2 情感真实性
- [ ] 情感反应符合情境
- [ ] 情感转变有铺垫
- [ ] 避免突兀的情绪变化

### 4.3 情感弧线
- [ ] 主角有清晰的情感发展
- [ ] 情感高潮与情节高潮匹配
- [ ] 结局情感状态合理

## 五、技术实现维度 (权重10%)

### 5.1 专业内容准确性
- [ ] 技术细节准确
- [ ] 专业术语使用正确
- [ ] 场景描写符合现实

### 5.2 逻辑自洽性
- [ ] 设定前后一致
- [ ] 规则应用一致
- [ ] 无自相矛盾

## 六、格式规范维度 (权重10%)

### 6.1 文档格式
- [ ] 章节标题格式统一
- [ ] 段落格式正确
- [ ] 无多余格式标记（如"** 数字"、"END OF CHAPTER"等）

### 6.2 文件组织
- [ ] 章节文件命名规范
- [ ] 文件编码正确（UTF-8）
- [ ] 无乱码或特殊字符

## 七、读者体验维度 (权重10%)

### 7.1 开篇吸引力
- [ ] 首章有吸引力
- [ ] 冲突或问题及时呈现
- [ ] 主角形象鲜明

### 7.2 节奏控制
- [ ] 张弛有度
- [ ] 无拖沓章节
- [ ] 高潮设置合理

### 7.3 结尾满意度
- [ ] 主要线索有交代
- [ ] 结局符合主题
- [ ] 有余韵或思考空间

## 评分标准

| 等级 | 分数范围 | 说明 |
|------|----------|------|
| A+ | 95-100 | 优秀，可直接发布 |
| A | 90-94 | 良好，轻微问题可选修 |
| B | 80-89 | 合格，建议修复中等问题 |
| C | 70-79 | 及格，需要修复问题 |
| D | 60-69 | 不及格，需要大幅修改 |
| F | <60 | 严重问题，需要重写 |

## 问题分级

### 严重问题（必须修复）
- 章节缺失或严重不完整
- 严重语法错误影响理解
- 情节逻辑断裂
- 角色设定矛盾
- 乱码或格式混乱

### 中等问题（建议修复）
- 模板化句式重复
- 情感描写模式化
- 词汇重复过多
- 格式标记问题

### 轻微问题（可选修复）
- 个别词汇可优化
- 句式可更多样化
- 细节可更丰富
