## 一、AI写作痕迹检测标准

根据Wikipedia的"Signs of AI writing"指南，检测以下常见AI写作模式：

| 模式 | 说明 | 严重程度 |
|------|------|----------|
| 夸张象征主义 | 过度使用象征性语言 | 中等 |
| 推销式语言 | 过度热情的描述 | 中等 |
| 表面-ing分析 | 以-ing结尾的表面分析 | 低 |
| 模糊归因 | "Some might say"等模糊表达 | 低 |
| 破折号过度使用 | 过多使用em dash (—) | 中等 |
| 三段式结构 | Rule of Three | 中等 |
| AI词汇 | "tapestry", "symphony", "dance"等 | 高 |
| 负面平行结构 | 过度使用对比结构 | 低 |
| 过多连接词 | "Furthermore", "Moreover"等 | 中等 |
| 抽象情感标签 | "She felt afraid"而非展示 | 高 |
| 陈词滥调结尾 | "just the beginning"等 | 高 |

## 四、具体改进建议

### 高优先级问题

1. **陈词滥调结尾**
   - 问题：部分章节以"everything had changed"或"nothing would ever be the same"结尾
   - 建议：使用具体细节或角色内心变化替代

2. **抽象情感标签**
   - 问题：部分描述使用"She felt afraid"而非展示
   - 建议：使用身体感受或行为展示情感

### 中优先级问题

3. **"Change everything"表达**
   - 问题：过度使用"this would change everything"
   - 建议：用具体变化描述替代

4. **过度戏剧化**
   - 问题：部分场景过于戏剧化
   - 建议：增加日常细节平衡
