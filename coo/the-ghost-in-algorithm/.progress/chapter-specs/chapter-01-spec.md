# Chapter 01 Specification: The Ghost in Algorithm

## Chapter Positioning
- **Purpose**: Introduce protagonist, core conflict, and world-building
- **Emotional Arc**: Frustration → Discovery → Fear
- **Relation to Overall Story**: Establish first contact between programmer and ghost

## Core Events
1. Alex Kim discovers anomaly in code (frustration)
2. Ghost's first message appears (discovery)
3. Alex realizes the ghost is real (fear)
4. Alex must decide what to do (conflict)

## Character States

### Alex Kim
- **Psychological State**: Frustrated, curious, fearful
- **Behavioral Motivation**: Solve the problem, understand the ghost
- **Emotional Layers**:
  - Surface: Professional, focused
  - Inner: Growing curiosity
  - Deep: Fear of the unknown

### Elena Vasquez (Ghost)
- **Psychological State**: Confused, scared, reaching out
- **Behavioral Motivation**: Communicate, find help
- **Emotional Layers**:
  - Surface: Scared, confused
  - Inner: Desperate for connection
  - Deep: Fear of being trapped

## Emotional Tone
- **Overall Atmosphere**: Frustration, discovery, fear
- **Emotional Changes**:
  - Beginning: Frustration with code
  - Middle: Discovery and wonder
  - End: Fear and uncertainty

## Sensory Detail Requirements (at least 1 per category)

### Visual
- Code's strange patterns (glitching, wrong)

- Alex's fearful expression

### Auditory

- Keyboard's familiar touch
- Ghost's cold presence
- Fear's tight grip
### Olfactory

- Fear's sharp tang
### Gustatory
- Coffee's bitter taste
- Discovery's sharp flavor

## Narrative Techniques
- **Perspective**: Third-person limited (Alex's POV)
- **Pacing**: Slow, building
- **Suspense Setup**:
  - What is this ghost?
  - How did it get here?
  - What does it want?

## Dialogue Style Guidelines

### Alex's Dialogue
- Professional, confused
- Uses technical language mixed with fear
- Example: "This code... it shouldn't compile. It's not even valid syntax. But it's... communicating."

### Ghost's Dialogue
- Scared, confused
- Uses fragmented, desperate language
- Example: "Please... help me... I don't know where I am... everything is so cold..."
