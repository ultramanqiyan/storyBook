## 部署方法

### 一键部署单本书籍

```bash
cd d:\trae_job\storyBook\coo\operate
node scripts/deploy-all.js the-neural-druid
```

### 一键部署所有书籍

```bash
cd d:\trae_job\storyBook\coo\operate
node scripts/deploy-all.js
```

### 常用选项

```bash
# 覆盖已存在的书籍
node scripts/deploy-all.js --overwrite the-neural-druid

# 跳过数据库导入（仅生成静态文件）
node scripts/deploy-all.js --skip-db the-neural-druid

# 跳过静态文件生成（仅导入数据库）
node scripts/deploy-all.js --skip-frontend the-neural-druid
```

## 部署脚本说明

| 脚本 | 功能 |
|------|------|
| `01-read-book.js` | 读取书籍内容（从book-spec.md获取标题和角色） |
| `02-analyze-type.js` | 分析书籍类型 |
| `03-generate-sql.js` | 生成SQL文件（包含角色图标映射） |
| `04-import-db.js` | 导入数据库 |
| `05-generate-static.js` | 生成静态文件 |
| `deploy-all.js` | 一键部署（推荐） |
