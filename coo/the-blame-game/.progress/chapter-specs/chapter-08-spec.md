# Chapter 8 规格：The Blame Game

## 章节定位
- **作用**：HR正式会议，Mark被要求"解释"
- **情绪曲线**：被背叛 → 绝望 → 无力
- **与整体故事的关系**：展示HR流程如何成为"合理化"工具

## 核心事件
1. 会议通知：Mark被"邀请"参加HR会议（被背叛）
2. 会议过程：Jennifer问"公平"的问题，Mark试图解释（绝望）
3. 意识到结局：Mark知道结论已经写好（无力）

## 角色状态

### Mark
- **心理状态**：绝望、无力、知道结局
- **行为动机**：为自己辩护，但知道徒劳
- **情感层次**：
  - 表面：冷静、专业
  - 内心：*This is a performance. The decision is already made.*
  - 深层：接受自己将成为替罪羊

### Jennifer Walsh（HR）
- **心理状态**：专业、执行流程
- **行为模式**：问"公平"问题、记录回答、保持中立
- **情感层次**：
  - 表面：中立、公平、"We want to hear your perspective"
  - 内心：*The process must be followed.*

### David Morrison
- **心理状态**：观察、确认
- **行为模式**：在场、偶尔补充信息
- **情感层次**：
  - 表面：专业、关心
  - 内心：*Almost done.*

### Michael Torres（法务）
- **心理状态**：专业、评估风险
- **行为模式**：在场、确保法律合规
- **情感层次**：
  - 表面：专业
  - 内心：*We need to document everything.*

## 叙事技巧
- **视角**：第三人称限制（Mark视角）
- **节奏**：缓慢、压抑，像一场预定的审判
- **悬念设置**：
  - HR会问什么？
  - Mark能为自己辩护吗？
  - 结论是什么？

## 对话风格指南

### Mark的对话
- 辩护、但知道徒劳
- 示例："I understand the concern. Let me provide some context..."
- 内心独白：*They're not listening. They've already decided.*

### Jennifer的对话
- 中立、公平、但冷
- 示例："Thank you for sharing that. We'll take everything into consideration."
- 特点：用"公平"的语言来合理化

### David的对话
- 补充、确认
- 示例："Just to clarify, Mark, you did sign off on the code review on March 20th, correct?"
- 特点：每个补充都在确认证据
