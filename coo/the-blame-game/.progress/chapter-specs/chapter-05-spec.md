# Chapter 5 规格：The Blame Game

## 章节定位
- **作用**：法务指出AI供应商合同免责条款，Mark的处境恶化
- **情绪曲线**：绝望 → 希望 → 更深的绝望
- **与整体故事的关系**：展示合同条款如何成为"保护伞"

## 核心事件
1. 法务介入：Michael Torres审查AI供应商合同（绝望）
2. 发现条款：合同明确将责任归于使用方（希望——也许供应商有责任？）
3. 条款反转：条款实际上保护了供应商，Mark的责任更大（更深的绝望）

## 角色状态

### Mark
- **心理状态**：从绝望到希望到更深的绝望
- **行为动机**：寻找任何可以转移责任的对象
- **情感层次**：
  - 表面：关注、期待
  - 内心：*Maybe the vendor is responsible. Maybe I'm not the one.*
  - 深层：知道自己在挣扎

### Michael Torres（法务）
- **心理状态**：专业、只关心法律风险
- **行为模式**：解读合同、定义责任边界
- **情感层次**：
  - 表面：专业、冷静
  - 内心：*The contract is clear. This is on us.*

### David Morrison
- **心理状态**：观察、记录
- **行为模式**：让法务说话
- **情感层次**：
  - 表面：关注、理解
  - 内心：*This is going exactly as expected.*

## 情感基调
- **整体氛围**：法律语言的冷酷、责任的明确化
- **情感变化**：
  - 开头：绝望
  - 中间：希望
  - 结尾：更深的绝望

  - 合同说了什么？

  - Mark还有出路吗？

## 对话风格指南

### Mark的对话
- 期待、寻找希望
- 示例："So the vendor might share some responsibility?"
- 内心独白：*Please let there be something.*

### Michael的对话
- 专业、法律性
- 示例："From a legal perspective, the contract explicitly states that the user assumes all responsibility for reviewing AI-generated code. SynthAI Corp is indemnified."
- 特点：只说法律事实，不评价

### David的对话
- 确认性
- 示例："So to clarify, the vendor has no liability in this situation?"
- 特点：确认"我们"的责任
