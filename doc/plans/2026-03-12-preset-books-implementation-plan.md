# 预设书籍数据实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task.

**Goal:** 为每种书籍类型创建2本预设书籍，共8本，每本包含角色、10个章节和谜题数据。

**Architecture:** 重写 `migrations/0002_seed_data.sql` 文件，包含完整的预设数据。每个章节从现有卡牌配置中选择4张卡牌（天气、地形、冒险、装备），并生成约300字的内容。

**Tech Stack:** SQL、SQLite
