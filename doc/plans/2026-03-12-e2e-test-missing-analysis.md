### 2.4 书籍创建页测试（book-create.spec.js）【详细版】

#### 2.4.1 页面功能概述

书籍创建页是一个**多步骤表单**，用户需要填写书籍基本信息、主角信息和配角信息，最终创建一本新书籍。

#### 2.4.2 用户交互流程图

```
┌─────────────────────────────────────────────────────────────────┐
│                      进入书籍创建页                              │
│                   (从书架页点击"创建新书")                        │
└─────────────────────────┬───────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                    步骤1：基本信息                               │
│  - 书籍名称（必填，最大50字符）                                  │
│  - 书籍类型（必选：儿童冒险/魔幻传说/都市言情/职场风云）          │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 验证：书名不为空，类型已选择
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                    步骤2：主角信息                               │
│  - 主角名称（必填，最大20字符）                                  │
│  - 选择头像（必选：预设emoji）                                   │
│  - 角色类型（必选：预设下拉 或 自定义输入）                       │
│  - 性格（必选：25种预设 或 自定义输入）                          │
│  - 说话方式（必选：25种预设 或 自定义输入）                       │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 验证：名称不为空，头像已选择，类型/性格/说话方式已填
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                    步骤3：配角信息（可选）                        │
│  - 配角数量：0-3个                                               │
│  - 每个配角：名称、头像、类型、性格、说话方式、亲密度、关系       │
└─────────────────────────┬───────────────────────────────────────┘
                          │ 验证：配角名称不为空，头像已选择
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                    步骤4：创建成功                               │
│  - 显示成功图标和提示                                            │
│  - 可跳转书籍详情页或返回书架                                    │
└─────────────────────────┬───────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────┐
│                    数据库写入                                    │
│  1. books表：创建书籍记录                                       │
│  2. characters表：创建主角记录（is_protagonist=1）              │
│  3. characters表：创建配角记录（is_protagonist=0）              │
└─────────────────────────────────────────────────────────────────┘
```
