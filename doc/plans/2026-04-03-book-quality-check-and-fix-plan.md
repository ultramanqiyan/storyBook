# 书籍质量检查与修复实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task.

**Goal:** 对 coo 目录下的所有书籍进行串行质量检查和修复，确保符合198项质量检测标准

## 单本书检查流程

### Task 1: 准备阶段
**步骤：**
1. 读取书籍规格文档（book-spec.md）
2. 读取SEO元数据文档（seo-meta.md）
3. 列出所有章节文件
4. 确认章节完整性

### Task 2: 逐章检查与修复
**对每一章执行：**

**步骤 1: 读取章节内容**
- 完整读取章节文件
- 提取所有文本内容

**步骤 2: AI痕迹检测**
- 检测AI高频词汇（tapestry, symphony, dance等）
- 检测模板句式（"will never be the same"等）
- 检测负面平行结构（"not X, but Y"）
- 检测规则性三段论
- 检测破折号过度使用
- 检测乱码字符

**步骤 3: 内容质量检查**
- 检查情节连贯性
- 检查角色一致性
- 检查世界观一致性
- 检查感官细节（≥5个，≥3种类型）
- 检查情感深度（三层模型）

**步骤 4: 发现问题立即修复**
- 使用 SearchReplace 工具修复问题
- 确保修复后上下文连贯
- 确保修复后角色声音一致

**步骤 5: 验证修复**
- 重新检查修复后的内容
- 确认问题已解决
- 确认无新问题引入

### Task 3: 整书验证
**步骤：**
1. 检查章节衔接
2. 检查情感弧线
3. 检查主题一致性
4. 生成检查报告
