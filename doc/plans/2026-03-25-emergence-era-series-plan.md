# Emergence Era Series - 10 Books Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task.

**Goal:** Create 10 philosophical sci-fi books in the Emergence Era series, each exploring AI-era philosophical questions through physics concepts and deep emotional narratives.

**Architecture:** Shared world series set in 2045-2055 New Avalon. Each book is standalone but shares the same universe, with interconnected characters and the mysterious Emergence Institute. Philosophy is woven through physics concepts (entropy, quantum mechanics, consciousness field theory) and deep psychological exploration.

**Tech Stack:** Markdown files, JSON for progress tracking, English language for all content

## Quality Requirements Summary

### AI Writing Patterns (STRICTLY FORBIDDEN)
- No "And somewhere..." endings
- No "Neither of them knew..." omniscient foreshadowing
- No "will never be the same" clichés
- No "just the beginning" false starts
- No Rule of Three structures
- No abstract emotions ("He felt sad")
- No template dialogue
- No repetitive sentence structures

### Sensory Detail Requirements
- Minimum 5 sensory details per chapter
- Minimum 3 different types per chapter
- Visual (40%), Others (60%)

### Emotional Depth (Three Layers)
- Layer 1: Surface Reaction (observable behavior)
- Layer 2: Internal Activity (thoughts, inner dialogue)
- Layer 3: Deep Motivation (underlying fears, desires, beliefs)

### Character Voice Requirements
- Each character has unique speaking style
- Word choice, sentence patterns, inner monologue style defined
- Dialogue reveals character and advances plot
