# 故事导演页UI重构设计

## 设计日期
2026-03-12

## 设计目标

| 必选槽位 | 金色边框高亮 |

## 五、扇形卡牌选择区设计

### CSS样式

```css
.fan-container {
  display: flex;
  align-items: flex-end;
  justify-content: center;
  min-height: 120px;
  position: relative;
}

.fan-card {
  width: 85px;
  height: 125px;
  position: absolute;
  bottom: 0;
  transform-origin: bottom center;
  background: linear-gradient(180deg, #2d2d44, #1a1a2e);
  border: 2px solid rgba(255, 215, 0, 0.3);
  border-radius: 10px;
}
```

### 扇形角度计算

- 最大展开角度：80度
- 角度计算：`angle = startAngle + angleStep * index`
- 水平偏移：`offsetX = (index - (count - 1) / 2) * 22px`

## 六、动画效果

| 动画 | 效果描述 | 触发条件 |
|------|----------|----------|
| 卡牌悬停 | 上浮50px + 放大1.25倍 + 发光阴影 | 鼠标悬停 |
| 选中状态 | 心跳动画 + 边框旋转发光 | 卡牌被选中 |
| 舞台震动 | 左右晃动反馈 | 选择卡牌时 |
| 天气特效 | 雨/雪/雷电粒子效果 | 选择天气后 |

## 九、实现要点

1. **布局重构**：将三栏布局调整为左右两侧卡牌选择区 + 中间舞台预览区
2. **扇形卡牌**：实现卡牌扇形展开效果，支持悬停和选中动画
3. **槽位网格**：8槽位网格布局，显示已选卡牌状态
4. **动画效果**：保留所有现有动画效果
5. **风格统一**：保持中世纪复古风格
