# 卡牌互动小说书籍网站 - UI展示和交互文档

## 文档信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 卡牌互动小说书籍网站 |
| 版本 | V1.0 |
| 创建日期 | 2026-03-11 |
| 文档类型 | UI设计文档 |

---

## 一、页面清单

| 页面名称 | 路径 | 说明 |
|----------|------|------|
| 首页/书籍广场 | `/` | 展示系统预设书籍 |
| 登录/注册页 | `/login.html` | 邮箱+密码登录/注册 |
| 书籍管理页 | `/bookshelf.html` | 用户书籍列表 |
| 书籍创建页 | `/book-create.html` | 多步骤创建书籍 |
| 书籍详情页 | `/book.html?id={id}` | 章节/角色列表 |
| 故事导演页 | `/director.html?bookId={id}` | 选择卡牌生成章节 |
| 章节阅读页 | `/chapter.html?id={id}` | 阅读章节+解谜 |

## 四、动画效果

### 4.1 卡牌动画

#### 悬停动画

```css
.card:hover {
  transform: perspective(1000px) rotateY(10deg) scale(1.05);
  box-shadow: 0 10px 30px rgba(0,0,0,0.3);
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}
```

#### 选中动画

```css
.card.selected {
  box-shadow: 0 0 20px rgba(255, 215, 0, 0.8);
  animation: heartbeat 1s infinite;
}

@keyframes heartbeat {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}
```

#### 获得卡牌动画

```css
@keyframes cardDrop {
  0% {
    transform: translateY(-100vh) rotate(0deg);
    opacity: 0;
  }
  50% {
    transform: translateY(20px) rotate(360deg);
    opacity: 1;
  }
  70% {
    transform: translateY(-10px);
  }
  100% {
    transform: translateY(0);
  }
}

.card-reward {
  animation: cardDrop 1s ease-out forwards;
}
```
