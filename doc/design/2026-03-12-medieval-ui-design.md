# StoryBook 中世纪复古奇幻风格 UI 设计方案

## 一、设计概述

### 核心风格定位

**中世纪复古奇幻风格**

融合炉石传说卡牌美学、古典羊皮纸/卷轴元素、暗色游戏主题，打造沉浸式的故事创作体验。

### 设计灵感来源

| 产品/游戏 | 借鉴元素 |
|-----------|----------|
| Hearthstone | 卡牌设计、金色边框、悬停动画 |
| The Witcher 3 | 暗色调、中世纪奇幻氛围 |
| Skyrim | 卷轴UI、羊皮纸背景 |
| D&D | 角色卡牌、属性系统 |

## 三、动画规范

### 入场动画

| 动画名称 | 时长 | 缓动函数 | 应用场景 |
|----------|------|----------|----------|
| fadeIn | 0.5s | ease-out | 页面元素淡入 |
| slideUp | 0.5s | ease-out | 元素从下滑入 |
| scaleIn | 0.3s | ease-out | 模态框缩放入场 |

### 交互动画

| 动画名称 | 时长 | 缓动函数 | 应用场景 |
|----------|------|----------|----------|
| cardHover | 0.3s | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 卡牌悬停上浮 |
| cardFlip3D | 0.8s | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 卡牌3D翻转 |
| scrollUnroll | 0.6s | ease-out | 卷轴展开 |
| pageFlip | 0.8s | ease-in-out | 书页翻动 |
| waxSeal | 0.3s | ease-out | 蜡封按下 |
| curtainOpen | 1s | ease-out | 幕布拉开 |

### 背景动画

| 动画名称 | 参数 | 应用场景 |
|----------|------|----------|
| particles | 50-100个粒子，15-25秒飘过 | 背景粒子效果 |
| shimmer | 3s infinite | 标题闪烁 |
| pulse | 2s infinite | 按钮脉冲 |

### 响应式动画策略

| 场景 | 桌面端 | 移动端 |
|------|--------|--------|
| 页面入场 | 完整动画 | 简化动画 |
| 卡牌悬停 | 3D翻转 | 简单放大 |
| 粒子背景 | 50-100个 | 20-30个 |
| 书籍抽出 | 完整3D效果 | 简单上浮 |

```
├── 特性展示区

│       ├── 角色卡牌
│       ├── 情节卡牌
│       └── 互动谜题
└── 预设故事区
    └── 书架堆叠展示
        ├── 扇形排列
        ├── 3D透视效果
        └── 悬停抽出动画
```

#### 设计细节

| 元素 | 样式 |
|------|------|
| 背景 | 渐变 `#1a1a2e → #16213e → #0f0f23` + 50-100个金色粒子 |
| 主标题 | Cinzel Bold 56px，金色渐变 `#FFD700 → #FFA500`，轻微阴影 |

| 书架书籍 | 扇形排列，每本书旋转角度不同，悬停抽出效果 |

#### 交互效果

| 交互 | 效果 |
|------|------|
| 卡牌悬停 | 上浮15px + 放大1.08倍 + 金色光晕 |
| 书籍悬停 | 抽出效果（上移30px + 放大1.1倍 + 取消旋转） |
| 按钮悬停 | 上浮5px + 发光增强 |

## 六、响应式设计

### 断点定义

| 断点 | 宽度范围 | 布局调整 |
|------|----------|----------|
| 移动端 | <768px | 单列布局，简化动画 |
| 平板 | 768-1024px | 双列布局，部分3D效果 |
| 桌面 | >1024px | 完整布局，全部效果 |

### 各页面响应式策略

| 页面 | 移动端 | 平板 | 桌面 |
|------|--------|------|------|
| 首页 | 单列，简化粒子 | 双列 | 完整布局 |
| 登录页 | 单页表单 | 居中卡片 | 书卷效果 |
| 书架页 | 单列网格 | 双列网格 | 三列网格 |
| 书籍详情页 | 单列，Tab切换 | 双栏 | 三栏 |
| 章节阅读页 | 全屏阅读 | 居中阅读 | 居中阅读 |
| 故事导演页 | 简化舞台 | 双栏 | 完整舞台 |
| 创建书籍页 | 全屏表单 | 居中表单 | 居中表单 |

## 七、可访问性

### 动画控制

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

### 对比度要求

| 元素 | 最小对比度 |
|------|-----------|
| 正文文字 | 4.5:1 |
| 大标题 | 3:1 |
| 图标 | 3:1 |

### 键盘导航

- 所有交互元素可通过Tab键访问
- 焦点状态清晰可见（金色边框）
- 模态框支持Escape键关闭

## 十、总结

本设计方案以**中世纪复古奇幻风格**为核心，融合炉石传说卡牌美学、古典羊皮纸/卷轴元素、暗色游戏主题，打造沉浸式的故事创作体验。

### 核心特点

1. **视觉一致性**：炉石卡牌风格贯穿全局
2. **沉浸感强**：羊皮纸、卷轴、蜡封等元素营造古典氛围
3. **交互精致**：丰富的动画效果，提升用户体验
4. **响应式设计**：适配桌面、平板、移动端
5. **可访问性**：支持减少动画偏好，键盘导航友好
