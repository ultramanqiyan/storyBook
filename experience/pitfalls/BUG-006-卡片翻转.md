# BUG-006 卡片翻转Bug

## 问题现象

人仔卡片翻转后，背面的按钮无法点击。

## 根本原因

CSS transform导致z-index失效。使用3D transform时，元素的层叠上下文发生变化，原有的z-index不再生效。

## 解决方案

```css
/* 修复前 */
.card:hover {
  transform: rotateY(180deg);
}

/* 修复后 */
.card:hover {
  transform: rotateY(180deg);
  z-index: 1000;
}

/* 或在父元素设置 */
.card-container {
  transform-style: preserve-3d;
}
```

## 预防措施

1. 使用3D transform时需要重新设置z-index
2. 理解transform对层叠上下文的影响
3. 测试翻转后的交互功能

## 相关代码

- `css/animations.css`
- `css/styles.css`

## 发现时间

2026-02-24

## 影响范围

所有使用3D卡片翻转的页面
