# BUG-008 数据库迁移Bug

## 问题现象

线上数据库结构与预期不符，新功能无法正常工作。

## 根本原因

迁移文件执行顺序问题。多个迁移文件没有版本号控制，导致执行顺序不确定，部分迁移未正确执行。

## 解决方案

1. 检查现有数据库结构
2. 按正确顺序执行迁移文件
3. 添加迁移版本控制

```sql
-- 迁移文件命名规范
migrations/
├── 0001_init.sql
├── 0002_add_puzzle_table.sql
└── 0003_add_share_table.sql
```

## 预防措施

1. 数据库迁移文件应有版本号，按顺序执行
2. 部署前检查数据库结构
3. 使用迁移工具管理版本

## 相关代码

- `migrations/`

## 发现时间

2026-02-25

## 影响范围

所有依赖数据库的功能

## 规则提炼

> 数据库迁移文件应有版本号，按顺序执行
