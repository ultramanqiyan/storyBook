# BP-002 Cloudflare D1 数据库操作

## 适用场景

在Cloudflare Pages Functions中操作D1数据库。

## 实践内容

### 参数化查询（防止SQL注入）

```javascript
// 正确做法 - 使用参数化查询
const result = await env.DB.prepare(
  'SELECT * FROM users WHERE user_id = ?'
).bind(userId).all();

// 错误做法 - 字符串拼接（有SQL注入风险）
const result = await env.DB.prepare(
  `SELECT * FROM users WHERE user_id = '${userId}'`
).all();
```

### 插入数据

```javascript
const result = await env.DB.prepare(`
  INSERT INTO users (user_id, username, created_at)
  VALUES (?, ?, datetime('now'))
`).bind(userId, username).run();
```

### 更新数据

```javascript
const result = await env.DB.prepare(`
  UPDATE books 
  SET title = ?, updated_at = datetime('now')
  WHERE book_id = ? AND user_id = ?
`).bind(title, bookId, userId).run();
```

### 查询单条记录

```javascript
const user = await env.DB.prepare(
  'SELECT * FROM users WHERE user_id = ?'
).bind(userId).first();
```

### 批量查询

```javascript
const result = await env.DB.prepare(
  'SELECT * FROM books WHERE user_id = ? ORDER BY created_at DESC'
).bind(userId).all();

const books = result.results;
```

## 注意事项

1. 始终使用参数化查询，防止SQL注入
2. 使用 `first()` 获取单条记录
3. 使用 `all()` 获取多条记录，结果在 `.results` 中
4. 使用 `run()` 执行插入/更新/删除
5. D1是SQLite，使用SQLite语法
