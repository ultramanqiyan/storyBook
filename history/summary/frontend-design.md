# 乐高故事书项目前端设计文档

## 文档说明

本文档面向前端开发人员和UI设计师。阅读本文档后，您将了解：
- 系统有哪些页面及其功能
- 每个页面的布局和交互设计
- 整体的视觉风格和设计规范
- 用户体验设计的考量

---

## 第一部分：页面概览

### 1.1 页面清单

| 页面名称 | 文件名 | 主要功能 | 用户访问频率 |

## 第二部分：各页面详细设计

### 2.1 登录页（login.html）

**页面目的**：让用户快速进入系统

**布局结构**：
```
┌─────────────────────────────────────────┐
│                                         │
│            [Logo图标]                    │
│           乐高故事书                     │
│                                         │
│     ┌─────────────────────────┐        │
│     │  请输入用户名            │        │
│     │  [________________]     │        │
│     │  0/20                   │        │
│     └─────────────────────────┘        │
│                                         │
│     ┌─────────────────────────┐        │
│     │      开始冒险 🚀         │        │
│     └─────────────────────────┘        │
│                                         │
│     ─────── 热门人仔 ───────           │
│     ┌────┐ ┌────┐ ┌────┐ ┌────┐       │
│     │骑士│ │巫师│ │公主│ │海盗│       │
│     └────┘ └────┘ └────┘ └────┘       │
│                                         │
└─────────────────────────────────────────┘
```

**交互说明**：

| 元素 | 交互行为 | 反馈 |
|------|----------|------|
| 用户名输入框 | 输入文字 | 实时显示字符数 |
| 开始冒险按钮 | 点击 | 按钮变灰，显示加载动画 |
| 热门人仔卡片 | 悬停 | 卡片放大，显示简介 |
| 热门人仔卡片 | 点击 | 快速选择该人仔形象 |

**业务规则**：
- 用户名为空时，按钮不可点击
- 用户名超过20字符时，无法继续输入
- 登录成功后，用户ID存储在浏览器本地存储

## 第四部分：交互设计规范

### 4.1 按钮设计

**主要按钮**：
- 背景：乐高黄(#FFD700)
- 文字：深色(#333)
- 悬停：背景加深10%
- 点击：缩小2%
- 禁用：透明度50%

**次要按钮**：
- 背景：透明
- 边框：乐高黄
- 文字：乐高黄
- 悬停：背景变为乐高黄，文字变深

### 4.2 卡片设计

**默认状态**：
- 背景：卡片背景色
- 阴影：轻微阴影
- 边框：无边框

**悬停状态**：
- 向上移动4px
- 阴影加深
- 边框：乐高黄边框

### 4.3 加载状态

**按钮加载**：
- 显示旋转动画
- 文字变为"加载中..."
- 禁用点击

**页面加载**：
- 显示全屏遮罩
- 中央显示旋转动画
- 下方显示提示文字

### 4.4 反馈提示

**成功提示**：
- 绿色背景
- 显示对勾图标
- 3秒后自动消失

**错误提示**：
- 红色背景
- 显示叉号图标
- 需要手动关闭

## 第五部分：响应式设计

### 5.1 断点设置

| 设备类型 | 断点 | 布局调整 |
|----------|------|----------|
| 手机 | < 768px | 单栏布局，隐藏侧边栏 |
| 平板 | 768px - 1024px | 两栏布局，缩小间距 |

**导航调整**：
- 顶部导航改为底部导航
- 侧边栏改为抽屉式
- 按钮全宽显示

**内容调整**：
- 字体适当放大
- 间距适当增加
- 卡片单列显示

## 第六部分：用户体验设计

### 6.1 新手引导

**首次登录引导**：
1. 欢迎弹窗介绍功能
2. 引导创建第一本书籍
3. 引导选择角色
4. 引导生成第一章

### 6.2 错误处理

**网络错误**：
- 显示友好的错误提示
- 提供重试按钮
- 保存用户已输入的内容

**AI生成失败**：
- 显示"AI服务暂时不可用"
- 提供稍后重试选项
- 不影响已生成的内容

### 6.3 无障碍设计

**键盘导航**：
- Tab键切换焦点
- Enter键确认操作
- Escape键关闭弹窗

**屏幕阅读器**：
- 为图片添加alt描述
- 为按钮添加aria-label
- 为表单添加label关联
