# 乐高故事书UI优化规格文档

## 一、优化目标

将现有页面改造为符合儿童审美的酷炫乐高风格界面，添加丰富的动画效果和明显的乐高元素。

## 二、设计原则

### 2.1 儿童友好设计
- 明亮、鲜艳的色彩
- 大按钮、易点击
- 清晰的视觉层次
- 有趣的交互反馈

### 2.2 乐高元素
- 乐高积木形状的按钮和卡片
- 乐高人仔风格的图标
- 积木堆叠效果
- 乐高经典配色（红、黄、蓝、绿）

### 2.3 动画效果
- 页面加载动画
- 按钮悬停效果
- 卡片翻转效果
- 积木堆叠动画
- 弹跳效果

## 三、色彩方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色-乐高红 | 红色 | #E3000F |
| 主色-乐高黄 | 黄色 | #FFD500 |
| 主色-乐高蓝 | 蓝色 | #006CB7 |
| 主色-乐高绿 | 绿色 | #00A651 |
| 背景-浅色 | 浅灰 | #F5F5F5 |
| 背景-深色 | 深蓝 | #1A1A2E |
| 强调色 | 橙色 | #FF6B35 |

## 四、组件设计

### 4.1 乐高按钮
- 3D立体效果
- 积木凸点装饰
- 悬停时弹跳动画
- 点击时下沉效果

### 4.2 乐高卡片
- 积木边框效果
- 悬停时浮起动画
- 阴影立体感
- 圆角积木形状

### 4.3 乐高导航
- 积木形状导航项
- 滑动高亮效果
- 悬停时放大动画

### 4.4 乐高人仔展示
- 圆形头像框
- 积木底座
- 弹跳入场动画

## 五、动画效果

### 5.1 页面加载
- 积木从天空掉落动画
- 依次堆叠效果
- 弹跳落地效果

### 5.2 按钮交互
- 悬停：放大1.05倍 + 阴影增强
- 点击：缩小0.95倍 + 下沉
- 释放：弹回原状

### 5.3 卡片交互
- 悬停：浮起 + 3D倾斜
- 点击：翻转效果

### 5.4 列表加载
- 依次入场动画
- 从左滑入 + 淡入

## 六、页面布局

### 6.1 首页
- 大型乐高积木背景
- 动态积木装饰
- 醒目的CTA按钮

### 6.2 书架页
- 积木书架效果
- 书籍卡片堆叠展示

### 6.3 人仔页
- 人仔卡片网格
- 积木底座展示

## 七、响应式设计

- 移动端：单列布局，大按钮
- 平板：双列布局
- 桌面：多列布局

## 八、性能考虑

- CSS动画优先
- 减少JavaScript动画
- 使用transform和opacity
- 避免重排重绘
