# 20个3D效果Demo规格文档

## 概述
基于用户要求，设计20个与现有5个3D效果差异明显的新Demo。每个Demo都有独特的视觉风格、交互方式和动画效果。

## 分类体系

### 几何变换类
- 06: 立方体盒子 - 6面体旋转展示
- 09: 魔方旋转 - 魔方拆解动画
- 15: 金字塔 - 埃及风格金字塔
- 17: 蜂巢网格 - 六边形蜂窝结构

### 物理模拟类
- 07: 螺旋星系 - 宇宙星系旋转
- 11: 球体轨道 - 行星公转系统
- 23: 龙卷风 - 螺旋上升气流
- 25: 弹簧动画 - 弹性物理效果

### 交互反馈类
- 08: 翻书效果 - 页面卷曲翻动
- 10: 卡片堆叠 - 扑克牌堆叠展开
- 12: 折叠面板 - 手风琴式展开
- 19: 环形菜单 - 圆形轨道菜单

### 空间穿梭类
- 13: 隧道穿梭 - 第一人称穿越
- 18: 瀑布流 - 3D卡片瀑布
- 21: 粒子爆炸 - 粒子扩散效果

### 特殊效果类
- 14: 波浪卡片 - 正弦波动画

- 20: 分形树 - 递归树形结构
- 22: 镜面反射 - 镜像对称效果
- 24: 多米诺骨牌 - 连锁反应动画

## 技术规格

### 通用要求
- 使用 CSS3 transform-style: preserve-3d
- 支持鼠标/触摸交互
- 流畅的60fps动画
- 响应式设计

### 差异化要点
1. 视觉风格：每个Demo使用独特的配色方案
2. 动画类型：旋转、平移、缩放、变形等不同组合
3. 交互方式：悬停、点击、拖拽、跟随等不同触发
4. 空间感：正面、透视、全景等不同视角
