## 统一设计系统

### CSS变量
```css
:root {
    --gold: #ffd700;
    --gold-dark: #b8860b;
    --silver: #c0c0c0;
    --crimson: #dc2626;
    --purple: #a855f7;
    --green: #22c55e;
    --blue: #3b82f6;
    --orange: #f59e0b;
    --bg-dark: #0d0d14;
    --bg-medium: #1a1a2e;
    --text-primary: #f8fafc;
    --text-secondary: #94a3b8;
    --font-title: 'Cinzel', serif;
    --font-body: 'Noto Sans SC', sans-serif;
}
```

### 通用组件样式

#### 按钮
- 主按钮：金色渐变 + 脉冲光晕
- 次要按钮：透明背景 + 白色边框
- 危险按钮：红色渐变

#### 卡片
- 背景：`linear-gradient(180deg, #2d2d44, #1a1a2e)`
- 边框：`2px solid rgba(255,255,255,0.1)`
- 悬停：金色边框 + 上浮效果

#### 炉石卡牌
- 尺寸比例：3:4
- 边框颜色按角色类型区分
- 悬停：上浮 + 放大 + 光晕

#### 输入框
- 背景：`rgba(0,0,0,0.4)`
- 边框：`2px solid rgba(255,215,0,0.3)`
- 聚焦：金色边框 + 光晕

#### 弹窗
- 背景：`linear-gradient(180deg, #2d2d44, #1a1a2e)`
- 边框：`3px solid var(--gold)`
- 遮罩：`rgba(0,0,0,0.8)` + 模糊

### 动画效果
- 粒子漂浮：20秒循环
- 卡牌悬停：上浮15px + 放大1.08倍
- 按钮脉冲：2秒循环光晕变化
- 弹窗出现：缩放0.9到1

## 实施顺序
1. 先创建通用CSS样式文件或在各页面添加样式
2. 按页面逐一改造
3. 测试响应式布局
4. 验证所有功能正常

## 注意事项
1. 保留所有现有API调用和功能逻辑
2. 只修改样式，不改变功能
3. 确保响应式设计正常工作
4. 测试深色/浅色主题切换（如有）
