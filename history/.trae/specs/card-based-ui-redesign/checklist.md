# 卡牌化UI设计 - 验收清单

## 一、卡牌组件验收

### 1.1 基础卡牌样式
- [ ] 卡牌尺寸符合设计规范（240px × 340px）

- [ ] 卡牌阴影效果正确

- [ ] 卡牌内容布局正确

### 1.2 卡牌翻转效果
- [ ] 翻转动画流畅（60fps）
- [ ] 翻转方向正确（Y轴）
- [ ] 翻转速度适中（0.8s）
- [ ] 正面内容完整显示
- [ ] 背面内容完整显示
- [ ] 翻转时无闪烁

### 1.3 稀有度系统
- [ ] 普通卡边框样式正确（白色）
- [ ] 稀有卡边框样式正确（蓝色发光）

- [ ] 稀有度宝石显示正确

### 1.4 角色类型标识

- [ ] 配角标识正确（银色边框+星星）

- [ ] 路人标识正确（灰色边框+简约）
- [ ] 类型徽章位置正确
- [ ] 类型徽章样式美观
