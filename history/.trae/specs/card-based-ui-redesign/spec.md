# 乐高故事书 - 卡牌化UI设计规范

## 📋 项目概述

### 目标
将现有的故事创建页面进行卡牌化改造，模仿炉石传说、三国杀等卡牌游戏的视觉风格和交互体验，让角色选择、情节选择等环节变成一张张精美的卡牌，提升用户的沉浸感和游戏化体验。

### 核心设计理念
- **卡牌化呈现**：所有选择项（角色、情节、天气、道具、地形等）都以卡牌形式展示
- **游戏化交互**：借鉴炉石传说的卡牌翻转、抽卡、发光等效果
- **稀有度系统**：引入稀有度概念，增加收集和探索乐趣
- **沉浸式体验**：通过精美的视觉设计和动画效果，让用户仿佛置身于卡牌游戏世界

## 🎯 推荐方案

### 主推荐：方案一（炉石传说风格）+ 方案三（乐高元素融合）

**理由：**
1. 炉石传说的卡牌设计成熟、视觉效果好，用户接受度高
2. 融合乐高元素可以增强品牌特色
3. 卡牌翻转、发光等效果能显著提升用户体验
4. 稀有度系统可以增加游戏的趣味性和收集欲望

### 具体实现建议：

1. **卡牌框架**：采用炉石风格的华丽边框设计
2. **角色形象**：使用乐高人仔形象，保持品牌一致性
3. **动画效果**：融合炉石的翻转效果和乐高的积木粒子
4. **色彩系统**：以乐高经典色为主，融入炉石的稀有度色系
5. **交互方式**：借鉴炉石的手牌区、战场区概念
