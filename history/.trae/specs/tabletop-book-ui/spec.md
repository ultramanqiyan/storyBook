# 炉石传说/桌游风格书籍内容展示页面规范

## 1. 项目概述

### 1.1 目标
创建一个全新的demo页面，将"图书管理员"风格彻底推翻，换成"炉石传说/桌游"风格的书籍内容展示界面。此页面为独立demo，不修改现有项目。

### 1.2 核心设计理念
- 书籍 = 一本巨大的魔法书/卡册
- 章节 = 一张张横向的任务卡/副本卡，整齐排列在桌面上
- 角色 = 竖立的随从卡牌，立在右侧的"卡槽"里
- 按钮 = 巨大的实体道具（如羽毛笔、卡包）
- 整体氛围 = 木质桌面、羊皮纸质感、厚重的边框、发光的宝石

---

## 2. 视觉设计规范

### 2.1 整体视觉基调 (Visual Theme)

#### 背景设计
- **主背景**: 深色橡木纹理桌面 (`#3e2723`)
- **备选背景**: 深绿色绒布赌桌 (`#1b5e20`)
- **装饰元素**: 角落散落的金币、骰子装饰
- **容器风格**: 卡册内页/游戏面板框架，厚重金属或雕刻木边框

#### 字体规范
- **标题字体**: 奇幻衬线体（类似《魔兽世界》或《炉石》标题字体）
- **文字效果**: 描边 + 投影
- **主字体**: Google Fonts - Cinzel (标题) + Noto Sans SC (正文)

### 2.2 色彩与材质规范

| 元素 | 材质/纹理 | 边框风格 | 配色方案 |
|------|----------|----------|----------|
| 背景 | 深色木纹/绿色绒布 | 无 | 深棕 `#3e2723` / 深绿 `#1b5e20` |
| 章节卡 | 泛黄羊皮纸/旧地图纹理 | 厚重金属/木雕 | 米黄 `#fff8e1` + 深褐边框 |
| 角色卡 | 光滑卡面/魔法光辉 | 六边形宝石框 | 依身份定（金/银/紫/木） |
| 按钮 | 魔法石/卡包/卷轴 | 发光符文 | 亮蓝/亮紫/金色 |
| 文字 | 奇幻衬线体 | 描边+投影 | 深黑/烫金/血红 |

## 3. 页面布局设计

### 3.1 顶部：书籍封面与属性栏 (Book Cover & Stats)

#### 左侧：立体书封
- 3D立体厚皮书效果
- 书脊上有金属铆钉
- 书名烫金印在封面上
- 鼠标悬停呼吸灯效果

#### 右侧：统计数据（宝石计数）
- 章节数 = 蓝色法力水晶堆叠 + 数字
- 角色数 = 金色金币堆叠 + 数字
- 字数 = 展开的卷轴长度 + 数字

### 3.2 左侧：章节任务板 (Chapter Quest Board)

#### 卡片设计（统一模板）
- **形状**: 横向长方形，四角金属包边
- **背景纹理**: 泛黄羊皮纸纹理，卷曲边缘效果
- **左侧**: 章节序号徽章（圆形蜡封印章/金属徽章，罗马数字）
- **中间**: 标题区域（丝带/横幅上，粗壮字体）
- **右侧**: 操作区
  - 编辑 = 小羽毛笔图标
  - 删除 = 小火把/碎纸机图标
  - 拖拽 = 卡片边缘抓手纹路
- **状态指示**:
  - 草稿 = 红色 "DRAFT" 印章
  - 完成 = 绿色 "DONE" 印章

#### 排列方式
- 卡片轻微间距，像洗好牌后摊开
- 拖拽时浮起并投下深深阴影

#### "添加章节"按钮
- **造型**: 背面朝上的空白卡牌/展开的空白卷轴
- **文字**: 中央巨大 "+" 号，周围魔法符文
- **动效**: 淡淡紫色雾气缭绕
- **点击反馈**: 卡牌翻转/卷轴展开

### 3.3 右侧：英雄与随从陈列架 (Hero & Minion Rack)

#### 卡片设计（统一模板）
- **形状**: 标准竖版卡牌比例（炉石卡牌风格）
- **边框颜色**:
  - 主角 = 金色龙纹边框
  - 配角 = 银色铁纹边框
  - 路人 = 木质普通边框
  - 反派 = 紫色邪能火焰边框
- **卡面内容**:
  - 上部: 角色立绘（60%）
  - 下部: 名字横幅
  - 底部: 两个宝石（登场次数）

#### 排列方式
- 卡牌略微重叠，扇形插在架子上
- 鼠标悬停浮起 + 发光特效（Legend特效）

#### "添加角色"按钮
- **造型**: 未开封卡包/空卡槽+问号+星光
- **动效**: 卡包微微颤动
- **点击反馈**: 卡包炸开金光

## 4. 交互体验规范

### 4.1 悬停效果 (Hover)
- 所有卡片放大 1.1 倍
- 向上浮起 `translateY(-10px)`
- 底部浓重黑色投影
- 边框流光扫过（Shine effect）

### 4.2 拖拽效果 (Drag)
- 鼠标指针变成"手"或"夹子"
- 被拖动卡片轻微旋转角度
- 其他卡片弹簧般缓冲动画让位

### 4.3 点击效果 (Click)
- 添加章节：翻转动画/撕裂音效
- 角色卡：卡牌敲击桌面声音

### 4.4 生成器弹窗
- 巨大法术书从屏幕中央翻开
- 或抽卡界面风格
- 背景变暗，聚光灯打在中间

## 5. 技术实现规范

### 5.1 文件结构
```
design-mockups/
  tabletop-book-demo.html    # 主demo文件（独立文件，不修改现有项目）
```

### 5.2 CSS动画要求
- 使用CSS3 transform实现卡片浮起效果
- 使用CSS animation实现流光、呼吸灯效果
- 使用CSS transition实现平滑过渡
- 支持`prefers-reduced-motion`媒体查询

### 5.3 响应式设计
- 桌面端：完整展示所有效果
- 平板端：适当缩小卡片尺寸
- 移动端：简化动画，保留核心视觉效果

### 5.4 性能优化
- 使用CSS transform代替position动画
- 使用will-change优化动画性能
- 图片使用CSS渐变和伪元素代替实际图片
